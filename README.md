# Palindrome

[![Build Status](https://travis-ci.org/CarlosBonetti/palindrome.svg?branch=master)](https://travis-ci.org/CarlosBonetti/palindrome)

A palindrome web app.

## How to run

To install the dependencies:

```
$ npm install
```

To initialize the app:

```
$ npm start
```

## Testing the app

```
$ npm test
```
